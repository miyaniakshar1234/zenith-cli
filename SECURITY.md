# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.1.x   | :white_check_mark: |
| 1.0.x   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within **Zenith CLI**, please do not open a public issue.

Instead, please contact the maintainer directly:
**Miyani Akshar** via GitHub [miyaniakshar1234](https://github.com/miyaniakshar1234).

We will prioritize the issue and release a patch as soon as possible.
